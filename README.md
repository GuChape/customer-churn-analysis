# Análise de Cancelamento de Clientes

## Descrição
Este projeto realiza uma análise de dados para identificar padrões e possíveis causas de cancelamento de clientes.

## Objetivo
Identificar padrões de comportamento que levam ao cancelamento de clientes e propor possíveis ações para redução do churn.

## Funcionalidades
- Limpeza e tratamento de dados
- Análise exploratória (EDA)
- Visualização com gráficos interativos
- Identificação de fatores que influenciam o cancelamento

## Tecnologias utilizadas
- Python
- Pandas
- Plotly

## Análise
O projeto analisa como diferentes variáveis impactam o cancelamento de clientes, utilizando gráficos para facilitar a interpretação dos dados.

## Como executar

1. Instale as dependências:
pip install -r requirements.txt

2. Execute o projeto:
python main.py


## Estrutura do projeto
- `main.py`: script principal da análise
- `cancelamentos_sample.csv`: base de dados utilizada

## Insights obtidos
- Clientes com contrato mensal apresentam maior taxa de cancelamento
- Alto número de ligações ao call center está associado ao churn
- Atrasos no pagamento influenciam diretamente o cancelamento
